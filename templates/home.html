GitGenius connects to your GitHub repositories, lets you choose the files that matter, and sends their contents to Groq's language models to draft comprehensive test suites. Instead of starting every scenario from scratch, you review AI suggestions and adapt them to your project structure.

  • Browse repositories and files directly through the GitHub OAuth integration.
  • Generate multi-language test cases powered by Groq's inference APIs.
  • Persist generated cases so you can revisit and refine them in the dashboard.
Enterprise-grade Reliability

The application relies on GitHub OAuth for authentication and keeps secrets outside the codebase via environment variables. Flask-Login manages user sessions while repository access stays scoped to the permissions you grant during sign-in.

  • OAuth 2.0 handshake handled by the dedicated GitHub service layer.
  • Configuration values such as client IDs and API keys live in your `.env` file.
  • Session and logout flows are powered by Flask-Login for predictable access control.
Always-on Groq Insights

Groq's API helps GitGenius deliver fast responses when you request new tests. Combined with the analytics models, you can track what has been generated and revisit results from the dashboard at any time.

  • Low-latency prompts through the Groq service keep generation snappy.
  • Analytics tables capture totals, technology breakdowns, and usage trends.
  • Access previously generated cases without rerunning the AI pipeline.
